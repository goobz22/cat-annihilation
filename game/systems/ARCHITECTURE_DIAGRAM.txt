 Player Input (UI)
      │
      ▼
 CatCustomizationSystem.setFurPattern(Tabby)
      │
      ├──▶ Update CatAppearance.pattern = Tabby
      │
      ▼
 Game Render Loop
      │
      ├──▶ getFurShaderParams()
      │         │
      │         └──▶ Build FurShaderParams struct
      │                   │
      │                   └──▶ Upload to GPU (UBO)
      │
      ├──▶ updateCatMaterial()
      │         │
      │         └──▶ Set Material.albedoColor = primaryColor
      │
      ▼
 Vulkan Render Pass
      │
      ├──▶ Bind cat_fur shader pipeline
      │
      ├──▶ Bind descriptor sets (camera, material, cat appearance)
      │
      ├──▶ Draw cat mesh
      │         │
      │         └──▶ cat_fur.frag executes
      │                   │
      │                   ├──▶ Generate pattern (procedural)
      │                   ├──▶ Apply fur lighting
      │                   └──▶ Output final color
      │
      ├──▶ For each equipped accessory:
      │         │
      │         ├──▶ getAccessoryTransform(slot)
      │         ├──▶ Render accessory model
      │         └──▶ Spawn particle effect (if any)
      │
      ▼
 Display to screen
