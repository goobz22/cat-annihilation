## Design Guidelines

- Each spell icon should reflect its element's color scheme
- Use particle effects and glowing edges to make icons pop
- Ultimate spells (level 10) should have more elaborate designs
- Include subtle animation frames if possible (4-8 frames)
- Size: 64x64 pixels
- Format: PNG with alpha channel
- Consider adding border glow matching element color

## Visual Hierarchy

- **Level 1 spells:** Simple, basic shapes
- **Level 3-5 spells:** Medium complexity with added effects
- **Level 7 spells:** Complex designs with multiple elements
- **Level 10 ultimates:** Most elaborate with special effects and larger visual impact
