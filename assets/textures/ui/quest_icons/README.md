# Quest Icons

This directory contains icons for different quest types in the Cat Annihilation game.

## Required Icons

1. **main.png** - Main story quest icon
   - Color: Gold (#FFD700)
   - Suggested design: Exclamation mark with crown or star
   - Size: 64x64 pixels

2. **side.png** - Side quest icon
   - Color: Silver (#C0C0C0)
   - Suggested design: Question mark or scroll
   - Size: 64x64 pixels

3. **daily.png** - Daily quest icon
   - Color: Light Blue (#4FC3F7)
   - Suggested design: Calendar or clock
   - Size: 64x64 pixels

4. **clan.png** - Clan mission icon
   - Color: Purple (#9C27B0)
   - Suggested design: Shield with clan emblem or paw print
   - Size: 64x64 pixels

5. **bounty.png** - Bounty quest icon
   - Color: Red (#F44336)
   - Suggested design: Skull or wanted poster
   - Size: 64x64 pixels

6. **default.png** - Default quest icon
   - Color: White (#FFFFFF)
   - Suggested design: Simple exclamation mark
   - Size: 64x64 pixels

## Design Guidelines

- Icons should be simple and recognizable at small sizes
- Use clear silhouettes with good contrast
- Include slight glow or border for visibility on various backgrounds
- Maintain consistent style with other UI elements
- Test visibility on both dark and light backgrounds
