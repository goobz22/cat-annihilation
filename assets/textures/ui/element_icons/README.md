# Element Icons

This directory contains icons for the four elemental types in the Cat Annihilation magic system.

## Required Icons

1. **water.png**
   - Color: Blue (#2196F3)
   - Suggested design: Water droplet, wave, or flowing water symbol
   - Theme: Healing and control

2. **air.png**
   - Color: White/Cyan (#E1F5FE)
   - Suggested design: Swirl, cloud, or wind gust
   - Theme: Speed and lightning

3. **earth.png**
   - Color: Brown/Green (#795548)
   - Suggested design: Mountain, rock, or crystal
   - Theme: Defense and durability

4. **fire.png**
   - Color: Red/Orange (#FF5722)
   - Suggested design: Flame, torch, or burning ember
   - Theme: Damage and destruction

## Usage

These icons are used in:
- Spellbook UI tabs
- Elemental progression displays
- Elemental affinity indicators
- Spell categorization

## Design Guidelines

- Icons should be instantly recognizable as their element
- Use strong, iconic shapes
- Include subtle glow effect in element's color
- Maintain visual balance across all four elements
- Size: 64x64 pixels recommended
- Format: PNG with alpha channel
- Can include subtle animation or particle effects
