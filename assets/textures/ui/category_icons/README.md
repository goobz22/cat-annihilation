# Category Icons

This directory contains icons for item category filters in the inventory UI.

## Required Icons

1. **weapon.png**
   - Design: Crossed swords or single sword
   - Color: Silver/Gray
   - Represents all weapon types
   - Size: 32x32 pixels

2. **armor.png**
   - Design: Shield or helmet
   - Color: Silver/Gray
   - Represents all armor pieces
   - Size: 32x32 pixels

3. **consumable.png**
   - Design: Potion bottle or food
   - Color: Red/Green
   - Represents consumable items
   - Size: 32x32 pixels

4. **material.png**
   - Design: Ore chunk or crafting icon
   - Color: Brown/Gray
   - Represents crafting materials
   - Size: 32x32 pixels

5. **quest.png**
   - Design: Scroll or quest marker
   - Color: Gold
   - Represents quest items
   - Size: 32x32 pixels

6. **misc.png**
   - Design: Bag or question mark
   - Color: Gray
   - Represents miscellaneous items
   - Size: 32x32 pixels

## Usage

These icons are used in:
- Inventory UI category filter buttons
- Item tooltips to show category
- Sorting and organization interfaces

## Design Guidelines

- Icons should be simple and instantly recognizable
- Use monochrome or limited color palette
- Maintain consistent size (32x32 recommended)
- Include subtle border or background
- Format: PNG with alpha channel
- Should work well in both selected and unselected states
- Consider creating highlighted versions for active filters
