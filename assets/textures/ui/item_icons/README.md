# Item Icons

This directory contains icons for all inventory items in the Cat Annihilation game.

## Naming Convention

Items should be named by their item ID:
- `sword_iron.png`
- `potion_health.png`
- `armor_leather_chest.png`
- `material_wood.png`
- etc.

## Item Categories

### Weapons
- Swords, bows, staves, claws, etc.
- Use distinctive silhouettes
- Color code by rarity (see below)

### Armor
- Helmets, chestpieces, gloves, boots, etc.
- Show the armor piece clearly
- Indicate material type through color/texture

### Consumables
- Potions, food, scrolls, etc.
- Use recognizable container shapes
- Color indicates potion type (red = health, blue = mana, etc.)

### Materials
- Crafting materials, resources, etc.
- Simple iconic representations
- Use realistic colors for materials

### Quest Items
- Unique quest-specific items
- Often have a glow or special effect
- Should stand out from regular items

### Miscellaneous
- Keys, tools, trinkets, etc.
- Varied designs based on item function

## Rarity Color Coding

Use border/glow colors to indicate rarity:
- **Common:** Gray (#CCCCCC)
- **Uncommon:** Green (#4CAF50)
- **Rare:** Blue (#2196F3)
- **Epic:** Purple (#9C27B0)
- **Legendary:** Orange (#FF9800)

## Design Guidelines

- Size: 64x64 pixels
- Format: PNG with alpha channel
- Use clear, recognizable silhouettes
- Include subtle shading for depth
- Add glow/border in rarity color
- Test visibility on inventory slot background
- Maintain consistent art style across all items
