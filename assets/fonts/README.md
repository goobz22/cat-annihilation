# Font Assets for Cat Annihilation

## Installed Fonts

### Open Sans
- **Files**:
  - `OpenSans-Regular.ttf` - Regular weight for body text
  - `OpenSans-Bold.ttf` - Bold weight for headers and emphasis

## Usage in Game

These fonts can be used for:
- UI text rendering
- In-game HUD elements
- Menu systems
- Score displays
- Wave notifications
- Dialogue text

## Technical Details

- **Format**: TrueType Font (.ttf)
- **File Size**: ~145KB each
- **Character Support**: Latin, Latin Extended, Cyrillic, Greek
- **Digitally Signed**: Yes

## Integration

For Three.js/React Three Fiber integration, use with:
- `THREE.FontLoader` for 3D text
- `troika-three-text` for performant 3D text rendering
- Standard DOM/CSS for UI overlays
